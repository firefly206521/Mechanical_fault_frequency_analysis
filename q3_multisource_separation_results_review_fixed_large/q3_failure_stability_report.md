# Q3 失败稳定性复验

## 结论

原先出现的三类失败并不都属于代码的确定性错误。

- 完全同频信号在 100 次大型测试中有 2 次误拆，正确不拆分率为 98%。把这两个失败样例的相位固定后，各更换 50 组新噪声，正确率分别为 94% 和 92%。因此误拆主要是低概率噪声假峰触发，不是某组相位必然导致的错误。
- 等幅、间隔 0.0025 Hz 在 200 次大型测试中成功率为 91%。18 次失败全部已经识别出两个分量，只是最大频率误差略高于验收线 Δf/4 = 0.000625 Hz。
- 不等幅、间隔 0.003 Hz 在 200 次大型测试中成功率为 87%。26 次失败中，20 次未保留弱分量，6 次识别出两个分量但频率误差略超验收线。

这些失败样例的设计矩阵条件数接近 1，不属于数值病态或 SVD 失效。

## 固定原失败相位、更换噪声

每个固定信号使用 50 组独立高斯噪声。没有一个目标条件达到“失败率不低于 90%”的稳定失败定义。

| 条件 | 最困难固定相位的成功率 | 主要失败形式 |
|---|---:|---|
| 完全同频 | 92% | 偶发误拆 |
| 等幅 0.0025 Hz | 26% | 两源均检出，但频率误差略超 Δf/4 |
| 不等幅 0.003 Hz | 32% | 弱源漏检或频率误差超限 |

这说明后两种条件位于概率性辨识边界：相位会显著改变难度，但噪声仍能改变单次结果。

## 相位网格搜索

在整个相位差区间搜索后，选择最困难的三个相位差，并分别用 100 组新噪声复验：

- 等幅 0.0025 Hz 的最坏相位差为 π，失败率 66%；
- 不等幅 0.003 Hz 的最坏相位差约为 2.618 rad，失败率 74%；
- 两者都没有达到 90% 的稳定失败门槛。

因此不能声称 0.0025 Hz 或 0.003 Hz 存在“每次稳定失败”的相位样例。它们是接近辨识极限后的统计性失败，而非固定代码路径错误。

## 稳定失败阳性对照

为验证诊断流程确实能够发现稳定失败，进一步采用更小频率间隔：

- 等幅 0.001 Hz，在相位差 0、π/2、π 下均为 50/50 失败；
- 不等幅 0.0015 Hz，在相位差 0、π/2 下均为 50/50 失败，在相位差 π 下为 49/50 失败。

这组结果与大型测试的辨识曲线一致，说明模型在明显低于 Rayleigh 分辨率时会稳定失效，而 0.0025 Hz 和 0.003 Hz 正处于由失败向成功过渡的区域。

- `q3_phase_grid_search.csv`：相位差粗网格搜索结果；
- `q3_phase_worst_case_validation.csv`：最坏相位的 100 次独立复验；
- `q3_stable_failure_controls.csv`：更小间隔下的稳定失败阳性对照。
