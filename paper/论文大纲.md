# 论文大纲：强噪声背景下机械设备微弱故障信号的检测、恢复与传感器布局优化

---

## 一、摘要

- 研究背景：机械设备故障产生的微弱周期信号淹没在强噪声中（SNR ≈ -12 dB）
- 核心问题链：检测 → 恢复 → 分离 → 布局优化
- 主要方法：GLRT 统计检验、谐波回归、多峰顺序检测 + BIC 定阶、鲁棒组合优化
- 关键结果：
  - Q1：GLRT 在 -12 dB 下检出 2 Hz 故障（统计量 2328.7 >> 阈值 25.1）
  - Q2：谐波回归恢复波形，Bootstrap 95% CI 频率 ±5.6×10⁻⁵ Hz
  - Q3：自动分离 4 个故障源，近频分辨极限达 0.002 Hz（0.8× Rayleigh）
  - Q4：最优三测点布局 bearing_left + gearbox_left + output_shaft，低 SNR 下 P_D 提升 2.3×

## 二、问题重述与分析

### 2.1 问题背景

齿轮/轴承在长期高负荷运转后会出现破损故障。当齿轮破损时，会在高速运转过程中产生具有特定频率的周期性冲击信号。机械电机本身的运转、传动系统的振动以及环境干扰共同产生强大的背景噪声，故障信号能量通常完全淹没在强噪声中。核心挑战是：如何从强噪声背景下有效检测并恢复出微弱的故障特征信号。

- 故障物理：破损齿轮 → 周期冲击 → 特征频率 f₀
- 噪声来源：电机运转 + 传动振动 + 环境干扰
- 信号条件：采样率 100 Hz，观测时长 400 s，样本数 40001，SNR 低至 -12 dB

### 2.2 四问递进逻辑

```
Q1: 有没有故障？  → 检测（频率存在性判定）
Q2: 故障长什么样？ → 恢复（完整波形参数估计）
Q3: 多个故障怎么办？→ 分离（多源盲辨识与定阶）
Q4: 传感器放哪最好？→ 优化（系统鲁棒性与布局决策）
```

## 四、第一问：基于 GLRT 的单源故障检测

### 4.1 信号模型

$$x(t) = A\sin(2\pi f_0 t + \phi_0) + n(t), \quad n(t) \sim \mathcal{N}(0,\sigma^2)$$

其中 $f_0$、$A$、$\phi_0$ 均未知，$\sigma^2$ 未知。

### 4.2 GLRT 检测框架

#### 4.2.1 二元假设

- $H_0$（无故障）：$x(t) = n(t)$，即 $A = 0$
- $H_1$（有故障）：$x(t) = A\sin(2\pi f_0 t + \phi_0) + n(t)$，即 $A > 0$

#### 4.2.2 似然比统计量

对每个候选频率 $f$，构造二维正交基矩阵：

$$X_f = \left[\sin(2\pi f \mathbf{t}),\; \cos(2\pi f \mathbf{t})\right]$$

GLRT 统计量（投影形式）：

$$T(f) = \frac{\mathbf{x}^T P_f \mathbf{x}}{\hat{\sigma}^2}, \quad P_f = X_f(X_f^T X_f)^{-1} X_f^T$$

其中 $\hat{\sigma}^2$ 为噪声方差的极大似然估计。

判决规则：

$$\max_{f \in \mathcal{F}} T(f) \;\; \overset{H_1}{\underset{H_0}{\gtrless}} \;\; \eta_\alpha$$

#### 4.2.3 与匹配滤波的等价性

在白噪声假设下，GLRT 与匹配滤波器在检测性能上等价。匹配滤波器是白噪声下已知波形的均匀最优势（UMP）检验，GLRT 将其推广到频率未知情形。

### 4.3 门限校准（Monte Carlo）

- 方法：生成 500 次纯高斯白噪声序列（长度 40001），每次计算 $\max_f T(f)$，取经验分布的 $(1-\alpha)$ 分位数
- 参数：$\alpha = 0.05$（目标虚警率），搜索范围 $0.05$–$49.5$ Hz
- 结果：$\eta_{0.05} \approx 25.10$

### 4.4 频率精修（变投影法）

FFT 粗搜索受栅格限制（$\Delta f = 0.0025$ Hz），在峰值邻域 $\pm 0.01$ Hz 内使用黄金分割搜索法最小化残差平方和，实现子毫赫兹精度。

精修目标函数：

$$\hat{f}_0 = \arg\min_{f} \min_{\alpha,\beta} \|\mathbf{x} - \alpha\sin(2\pi f\mathbf{t}) - \beta\cos(2\pi f\mathbf{t})\|^2$$

### 4.5 实验结果

#### 4.5.1 实际数据检测

| 指标 | 数值 |
|---|---|
| GLRT 统计量最大值 | **2328.66** |
| Monte Carlo 阈值 ($\alpha=0.05$) | **25.10** |
| 判定 | **检出（统计极显著）** |
| 精修频率 | **2.000001535 Hz** |
| 拟合振幅 | 0.03519 |
| 拟合相位 | 1.57365 rad |
| 残差标准差 | 0.10001 |
| 解释方差 | 5.83% |
| 估计输入 SNR | **-12.08 dB** |
| 2 Hz 峰值抑制 | 30.45 dB（移除分量后） |

#### 4.5.2 分段稳定性验证

8 个 50 秒分段全部检出约 2 Hz 故障频率，频率偏差均 < 0.001 Hz，表明故障分量在全时段稳定存在。

#### 4.5.3 仿真对比（FFT 峰值法 vs GLRT）

| SNR (dB) | FFT P_D | GLRT P_D | FFT 频率 MAE (Hz) | GLRT 频率 MAE (Hz) |
|---|---|---|---|---|
| -25 | 83.75% | **97.50%** | 3.90 | **0.27** |
| -22 | 100% | 100% | — | — |

在极低 SNR 下 GLRT 显著优于 FFT 峰值法。

### 4.6 对照模型

| 模型 | 检出频率 | 特点 |
|---|---|---|
| FFT 峰值法 | ~2 Hz | 简单基线，低 SNR 下频率误差大 |
| Welch PSD | ~2 Hz | 方差缩减，牺牲频率分辨率 |
| CFAR + FFT | ~2 Hz | 自适应门限，不需先验噪声水平 |
| SSA + FFT | ~2 Hz | 去噪后检测，依赖窗口选择 |
| MUSIC | ~2 Hz | 超分辨，计算量大（适合 Q3 对照） |

GLRT 推荐为主模型的理由：可控误报率、坚实统计基础、可解释判决量、论文友好。

## 五、第二问：基于谐波回归的单源波形恢复

### 5.1 从检测到恢复

Q1 给出了 $f_0 \approx 2$ Hz 的可靠检测。Q2 的目标是在此基础上估计完整的正弦参数 $(f_0, A, \phi_0)$，从而恢复故障信号的时域波形。

### 5.2 谐波回归模型

#### 5.2.1 线性化

将正弦信号改写为同频 sine/cosine 的线性组合：

$$s(t) = A\sin(2\pi f_0 t + \phi_0) = \alpha\sin(2\pi f_0 t) + \beta\cos(2\pi f_0 t)$$

其中 $\alpha = A\cos\phi_0$，$\beta = A\sin\phi_0$。

#### 5.2.2 参数估计

- 固定 $f_0$，用线性最小二乘求解 $\hat{\alpha}, \hat{\beta}$
- 极坐标转换：$\hat{A} = \sqrt{\hat{\alpha}^2 + \hat{\beta}^2}$，$\hat{\phi}_0 = \text{atan2}(\hat{\beta}, \hat{\alpha})$
- 再对 $f_0$ 用黄金分割搜索（55 次迭代）最小化残差 SSE

#### 5.2.3 最终恢复波形

$$\hat{s}(t) = 0.035190317 \cdot \sin(2\pi \times 2.0000016152 \cdot t + 1.573555072)$$

| 参数 | 估计值 |
|---|---|
| 频率 $f_0$ | 2.0000016152 Hz |
| 振幅 $A$ | 0.035190317 |
| 初相位 $\phi_0$ | 1.573555072 rad |
| 残差标准差 | 0.100013640 |
| 解释方差 | 5.8294% |
| 估计输入 SNR | -12.083 dB |

### 5.3 不确定性量化（Bootstrap）

对残差重采样 10000 次，重复谐波回归，得到参数经验分布：

| 参数 | 点估计 | 95% 置信区间 | Bootstrap 标准差 |
|---|---|---|---|
| 频率 (Hz) | 2.00000162 | [1.99994563, 2.00005589] | 2.8 × 10⁻⁵ |
| 振幅 | 0.0351903172 | [0.033815617, 0.036562463] | 0.000707 |
| 相位 (rad) | 1.57355507 | [1.49533205, 1.65507626] | 0.0408 |

频率 95% CI 宽度仅 $1.1 \times 10^{-4}$ Hz，精度极高。

## 六、第三问：基于顺序 GLRT 的多源盲分离

### 6.1 从单源到多源

信号模型扩展为多正弦叠加：

$$x(t) = \sum_{k=1}^{K} A_k\sin(2\pi f_k t + \phi_k) + n(t), \quad n(t) \sim \mathcal{N}(0,\sigma^2)$$

其中 $K$、$\{f_k\}$、$\{A_k\}$、$\{\phi_k\}$ 全部未知。挑战在于：（1）自动确定源数 $K$；（2）在频率可能接近时准确分离各分量。

### 6.2 多峰顺序检测算法

```
1. 初始化：空模型，残差 = 去线性趋势后的信号
2. 循环：
   a. 对当前残差计算 GLRT 统计量 T(f)
   b. 若 max T(f) < η_α：停止
   c. 提取峰值频率作为候选 f_new
   d. 若 f_new 与已有频率靠近（< 0.02 Hz）：
      → 调用 close_pair_resolver 尝试拆峰
   e. 以当前所有候选频率进行多频联合最小二乘拟合（SVD）
   f. 计算 BIC，若 ΔBIC ≥ 10 且设计矩阵良态：保留
      否则：拒绝
   g. 更新残差，回到 (a)
3. 输出：最终分量集合
```

### 6.3 BIC 模型定阶

$$\text{BIC}(K) = N \ln(\hat{\sigma}^2_K) + (3K+1)\ln N$$

- 每分量 3 个自由参数（$f_k, A_k, \phi_k$），+1 为 $\sigma^2$
- ΔBIC ≥ 10 视为强证据接受新分量（对应后验概率比 > 150:1）
- 联合拟合使用 SVD 最小二乘，条件数超过 10⁶ 时标记为不可靠分离

### 6.4 近频分辨机制

当候选频率与已有频率间距 < 0.02 Hz 时，调用 close_pair_resolver：

1. **下变频到基带**：混频信号乘以复指数 $\exp(-j2\pi f_c t)$，以两频率中点 $f_c$ 为中心
2. **局域周期图**：0.000025 Hz 步长扫描，定位最强分量
3. **剥离-重扫描**：减去最强分量后扫描残差中的次强分量
4. **二维变投影联合精修**：双频率同时优化
5. **BIC 裁定**：双源模型 vs 单源模型的 BIC 比较

## 七、第四问：鲁棒传感器布局优化

### 7.1 从信号处理到系统设计

前三问解决了"给定测点信号 → 检测/恢复/分离故障"的信号处理问题。Q4 在此基础上提出系统工程问题：在设备上布置有限数量（≤3）的加速度传感器，使整个检测系统对微弱故障的综合检测概率最大化。

### 7.2 多传感器观测模型

$$y_m(t) = \sum_{k=1}^{K} h_{mk} A_k \sin(2\pi f_k t + \phi_k + \delta_{mk}) + n_m(t), \quad m = 1,\ldots,M$$

- $h_{mk}$：第 m 个测点对第 k 个故障源的综合敏感度（含传播衰减和测点响应差异）
- $\delta_{mk}$：传播路径引入的相位偏移
- $n_m(t)$：第 m 个测点的本地噪声，方差 $\sigma_m^2$

### 7.3 融合检测策略

各测点独立计算 GLRT 统计量 $T_m(f)$，按逆噪声方差加权融合：

$$T_{\text{fused}}(f) = \sum_{m \in S} w_m T_m(f), \quad w_m = \frac{1/\hat{\sigma}_m^2}{\sum_{j \in S} 1/\hat{\sigma}_j^2}$$

噪声大的测点贡献小，噪声小的测点贡献大，实现最优信息融合。

### 7.4 布局评分函数

$$J(S) = \overline{P_D}(S) - \lambda \cdot P_{FA}(S) - \mu \cdot \operatorname*{Var}_{k}\left(P_{D,k}(S)\right)$$

三项含义：
1. **$\overline{P_D}(S)$**（最大化）：所有故障源在所有 SNR 条件下的平均检测概率
2. **$-\lambda P_{FA}(S)$**（惩罚虚警）：$\lambda = 2.0$，虚警率每增加 1%，评分扣 0.02
3. **$-\mu \operatorname{Var}_k(P_{D,k}(S))$**（惩罚不均衡）：$\mu = 0.5$，防止布局"偏爱强源、忽视弱源"

### 7.5 候选测点与敏感度矩阵

6 个工程可行的加速度传感器安装位置：

| 测点 | 4 Hz 源 | 8 Hz 源 | 13 Hz 源 | 14 Hz 源 | 物理依据 |
|---|---|---|---|---|---|
| bearing_left | 0.95 | 0.65 | 0.35 | 0.45 | 近轴承，对低频敏感 |
| bearing_right | 0.60 | 0.95 | 0.45 | 0.50 | 近轴承，对中频敏感 |
| gearbox_left | 0.45 | 0.55 | 1.05 | 0.75 | 近齿轮箱，对高频敏感 |
| gearbox_right | 0.40 | 0.50 | 0.80 | 1.10 | 近齿轮箱，14 Hz 最强响应 |
| input_shaft | 1.10 | 0.45 | 0.35 | 0.25 | 输入端，4 Hz 最强响应 |
| output_shaft | 0.35 | 0.50 | 0.65 | 1.00 | 输出端，14 Hz 强响应 |

故障频率和振幅沿用 Q3 结果（4 个故障源）。

### 7.6 优化与仿真设计

- 搜索空间：C(6,1) + C(6,2) + C(6,3) = 6 + 15 + 20 = **41 种布局**
- 4 个鲁棒性测试场景：均衡（balanced）、噪声偏斜（spatial_noise_skew）、局部失敏（weak_single_sensor）、传播衰减（source_shadow）

#### 7.7.1 最优布局

**三传感器最优布局**：**bearing_left + gearbox_left + output_shaft**

在基线实验中排名第一，评分 0.6165，在 6 组不同参数实验和 5 个随机种子下均位列前二。

### 7.8 布局策略原理分析

最优布局 **bearing_left + gearbox_left + output_shaft** 的优势源于：

1. **互补敏感度**：bearing_left 覆盖 4 Hz（0.95）和 8 Hz（0.65），gearbox_left 覆盖 13 Hz（1.05）和 14 Hz（0.75），output_shaft 补充 14 Hz（1.00）和 13 Hz（0.65）
2. **冗余与鲁棒**：每个故障源至少有 2 个测点具有中等以上敏感度
3. **空间分散**：三测点分布在设备不同位置，对局部噪声和传播衰减有天然鲁棒性
4. **避免过度冗余**：评分函数中的 $\mu \operatorname{Var}_k$ 项惩罚"所有测点都对同一强源最敏感"的布局

### 7.9 V1 扩展：自适应连续空间规划

V0 方案从 6 个离散候选点选择。V1 修订方案作为方法论扩展：

1. 在可安装表面生成粗到细的自适应网格（不预设固定测点）
2. 使用降阶模态模型或数据驱动传播模型表示结构传播
3. **最大最小目标**：$\max_{S,|S|\le 3} \min_{k,\theta\in\Theta} P_D(\lambda_k(S;\theta), \alpha)$，即优先保护最弱故障源在最坏不确定参数下的检测概率
4. 分级计算策略：快速筛选（非中心参数/信息矩阵）→ 组合优化（贪心+交换）→ 最终 GLRT+Monte Carlo 验证（仅对 5–10 个候选布局）

## 八、模型评价与讨论

### 8.1 方法优势

| 维度 | 优势 |
|---|---|
| **理论基础** | GLRT 提供严格统计检验框架，可控 I 类错误率；BIC 提供一致模型选择准则 |
| **参数精度** | 变投影精修 + 联合拟合实现子毫赫兹频率精度；Bootstrap 提供参数不确定性量化 |
| **自动化** | 多峰顺序检测 + BIC 自动确定故障源数量；不需人工判断"几个峰" |
| **超分辨率** | 近频分辨达 0.8–1.2× Rayleigh 极限，超越传统 FFT 分辨能力 |
| **系统工程** | 传感器布局优化将信号处理与硬件部署统一考虑，提供可操作工程建议 |
| **可复现性** | 所有结论经 Monte Carlo 仿真（200–500 次）+ 多随机种子 + 多场景交叉验证 |

### 8.2 局限与改进方向

| 局限 | 改进方向 |
|---|---|
| 依赖高斯白噪声假设 | 残差诊断基本支持，但可扩展至有色噪声的广义 GLRT |

| 未利用故障谐波结构 | 若齿轮齿数已知，可引入基频 × 齿数的谐波约束，提升检测灵敏度 |
| 传播模型简化为静态增益 | V1 已提出模态模型扩展；需真实结构动力学参数或实验标定 |
| 纯仿真验证 | 建议补充振动台实验或公开数据集验证（如 CWRU 轴承数据集） |
| 计算开销大（Q4 过夜运行） | V1 分级计算策略可显著减少全规模 GLRT 仿真次数 |

## 九、结论

本研究针对强噪声背景下机械设备微弱故障信号的四层次问题，给出了系统性的数学建模和求解方案：

1. **故障检测（Q1）**：GLRT 统计检验在 SNR ≈ -12 dB 下成功检出 2.0000015 Hz 故障频率，检验统计量（2328.7）远超 Monte Carlo 校准门限（25.1），统计极显著。在 -25 dB 的极低 SNR 合成实验中，GLRT 检测概率（97.5%）显著优于 FFT 峰值法（83.75%）。

2. **波形恢复（Q2）**：谐波回归精确恢复了故障信号波形参数 $f_0 = 2.0000016152$ Hz、$A = 0.03519$、$\phi_0 = 1.5736$ rad。Bootstrap 95% 置信区间极窄（频率宽度 $1.1\times10^{-4}$ Hz），SSA 对照相关系数 0.9956。残差诊断支持高斯白噪声假设。

3. **多源分离（Q3）**：多峰顺序 GLRT + BIC 自动定阶成功分离出 4 个故障源（4/8/13/14 Hz），联合解释方差 11.92%。近频分辨极限达 0.002 Hz（0.8× Rayleigh 极限），SNR ≥ -12 dB 时 K 识别正确率 ≥ 98%，纯噪声误报率仅 0.2%。

4. **传感器布局（Q4）**：推荐的最优三测点布局为 **bearing_left + gearbox_left + output_shaft**，在 -25 dB 极低 SNR 下检测概率达 19.2%，相较单最优测点提升 2.3 倍，达到全六测点参考性能的 84%。结论经 4 个鲁棒性场景、6 组参数、5 个随机种子交叉验证，高度稳健。

### 附录 C：关键算法伪代码

#### C.1 GLRT 检测（Q1）

```
输入: 信号 x[n], 采样率 fs, 目标虚警率 α
输出: 是否检出, 精修频率 f0_hat, GLRT 统计量

1. x = detrend(x)                          // 线性去趋势
2. for f in search_range(0.05, 49.5, Δf):  // 粗搜索
       T[f] = GLRT_statistic(x, f, fs)     // 投影统计量
3. η = monte_carlo_threshold(α, N_mc=500)  // 纯噪声门限校准
4. if max(T) > η:
       f_coarse = argmax(T)
       f0_hat = golden_section_refine(x, f_coarse, fs)  // 精修
       detected = True
   else:
       detected = False
5. return detected, f0_hat, max(T)
```

#### C.2 多峰顺序检测（Q3）

```
输入: 信号 x[n], 采样率 fs, α, ΔBIC_min=10
输出: 分量列表 [(f_k, A_k, φ_k)], K

1. components = []
2. residual = detrend(x)
3. while True:
       T_f = GLRT_scan(residual, fs)
       if max(T_f) < η:
           break
       f_candidate = argmax(T_f)
       if near_existing(f_candidate, components):      // < 0.02 Hz
           f_candidates = close_pair_resolve(residual, f_candidate, components, fs)
       else:
           f_candidates = [f_candidate] + [c.f for c in components]
       α_hat, β_hat = joint_LS_fit_SVD(x, f_candidates, fs)
       if cond(X) < 1e6 and BIC_improvement(α_hat, β_hat, residual) >= ΔBIC_min:
           components = update(α_hat, β_hat, f_candidates)
       else:
           break
       residual = x - reconstruct(components, fs)
4. return components, len(components)
```

#### C.3 传感器布局优化（Q4）

```
输入: 候选测点集合, 敏感度矩阵 H, 故障源参数, 评分权重 λ, μ
输出: 最优布局 S*

1. all_layouts = [C(M,1) ∪ C(M,2) ∪ C(M,3)]           // 共 41 种
2. for S in all_layouts:
       for scenario in [balanced, noise_skew, weak_sensor, shadow]:
           for snr in SNR_range:
               P_D[k][snr] = monte_carlo_GLRT_fused(S, H, scenario, snr)
               P_FA = monte_carlo_false_alarm(S, H, scenario)
       J(S) = mean(P_D) - λ × P_FA - μ × var_k(P_D[k])
3. S* = argmax_S J(S)
4. return S*
```
