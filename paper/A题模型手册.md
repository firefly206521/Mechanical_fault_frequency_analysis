# A题：机械设备故障检测 — 模型手册

> 本文档汇总所有候选模型的原理、步骤与文献依据，不包含主观评分。每个模型独立描述，供编程实现和实验对比使用。

---

## 数据概况

- 采样频率：$f_s = 100$ Hz（采样间隔 0.01 s）
- 采样点数：$N = 40001$
- 观测时长：$T = 400$ s
- 基础频率分辨率：$\Delta f = f_s / N = 0.0025$ Hz
- 单源故障 SNR：约 -12.1 dB
- 多故障源 SNR：总计约 -8.7 dB
- 噪声特性：残差近似高斯白噪声

---

## 信号模型

**单源故障**：

$$x(t) = A \sin(2\pi f_0 t + \varphi_0) + n(t)$$

**多源故障**：

$$x(t) = \sum_{k=1}^{K} A_k \sin(2\pi f_k t + \varphi_k) + n(t)$$

其中 $n(t)$ 为强电机噪声，故障信号能量远小于噪声能量。

## 第一部分：频域类模型

### 1. 功率谱分析 + CFAR 门限 + 谐波回归

**类别**：频域检测 + 时域参数估计

**核心原理**：

- 对加窗信号做周期图（Periodogram）得到功率谱密度估计
- 正弦信号在频谱上表现为单一尖峰，噪声能量散布全频带
- CFAR（恒虚警率）门限：用局部频谱中位数估计噪声底板，设置自适应门限判断峰值是否显著
- 以频谱峰为初值做非线性最小二乘，精修频率；再用谐波回归估计振幅和初相位

**关键公式**：

加窗周期图：

$$P(f) = \left| \sum_{n=0}^{N-1} w_n x_n e^{-j 2\pi f n / f_s} \right|^2$$

CFAR 判决：

$$P(f) > \gamma \cdot \widehat{P}_{\text{noise}}(f)$$

其中 $\widehat{P}_{\text{noise}}(f)$ 为 $f$ 附近局部频谱中位数，$\gamma$ 由目标虚警率确定。

频率精修（可变投影 / 非线性最小二乘）：

$$\hat{f} = \arg\min_f \min_{a,b,c} \sum_{n=1}^{N} \left[ x_n - a\sin(2\pi f t_n) - b\cos(2\pi f t_n) - c \right]^2$$

谐波回归恢复波形：

$$x_n = a\sin(2\pi \hat{f} t_n) + b\cos(2\pi \hat{f} t_n) + c + \varepsilon_n$$

$$\hat{A} = \sqrt{a^2 + b^2}, \quad \hat{\varphi} = \operatorname{atan2}(b, a)$$

**实现步骤**：

1. 数据预处理：去均值、去线性趋势、乘 Hann 窗
2. FFT 计算功率谱
3. 估计局部噪声底板（滑动中值滤波）
4. 对超出 CFAR 门限的峰值排序，取最高峰作为候选 $f_0$
5. 以候选 $f_0$ 为初值，做可变投影（外层优化 $f$，内层闭式求解 $a,b,c$）
6. 固定 $\hat{f}$，用谐波回归求 $\hat{A}$ 和 $\hat{\varphi}$

**文献中的优点**：

- 结构简单，数值稳定
- 长观测时间（400 s）提供强相干积累，功率谱峰显著
- 频率估计和波形恢复共享同一组参数

**文献中的局限**：

- 两频率间隔 $< \Delta f$ 时峰值可能重叠
- CFAR 门限需要通过理论（卡方分布）或仿真确定

**适用问题**：Q1（检测 + 频率估计）、Q2（波形恢复，直接输出 $A, \varphi$）

### 2. 包络谱分析

**类别**：频域解调分析

**核心原理**：

- 齿轮故障产生的冲击信号对高频载波（结构共振）进行调幅
- Hilbert 变换求解析信号 → 取模得包络 → 对包络做 FFT → 包络谱
- 包络谱中的峰值对应冲击重复频率（即故障特征频率）
- 该方法在旋转机械故障诊断文献中是标准手段 [Randall et al., 2001]

**关键公式**：

解析信号：

$$x_a(t) = x(t) + j \cdot \mathcal{H}\{x(t)\}$$

包络：

$$e(t) = |x_a(t)| = \sqrt{x^2(t) + \mathcal{H}\{x(t)\}^2}$$

包络谱：

$$E(f) = \left| \text{FFT}\{e(t)\} \right|$$

**实现步骤**：

1. 对 $x(t)$ 做 Hilbert 变换
2. 构造解析信号，取绝对值得到包络
3. 去除包络的均值
4. 对包络做 FFT，得到包络谱
5. 在包络谱中搜索峰值频率

**文献中的优点**：

- 对调幅冲击信号比直接 FFT 更敏感
- 物理可解释性强（冲击重复率直接对应故障频率）
- 工程验证充分，文献支撑丰富

**文献中的局限**：

- 依赖信号具有调幅冲击结构（纯正弦信号效果等同于直接 FFT）
- 若解调频带选择不当，效果显著下降 [Smith et al., 2019]
- 强噪声下包络可能仍被噪声主导

**适用问题**：Q1（频率检测的可视化对照方法）、Q3（多源分离的支撑工具）

### 3. 谱峭度（Spectral Kurtosis）与 Kurtogram

**类别**：频域 + 高阶统计量

**核心原理**：

- 谱峭度衡量每个频带的**冲击性**程度（峭度 = 四阶标准化矩）
- Kurtogram：在频率-带宽平面上画出谱峭度分布图，自动定位"冲击最强"的频带
- 找到最佳频带后，在该频带内做滤波 + 包络解调

**关键公式**：

谱峭度定义为复包络 $H(n, f)$ 的四阶标准化累积量：

$$\mathcal{K}(f) = \frac{\langle |H(n, f)|^4 \rangle}{\langle |H(n, f)|^2 \rangle^2} - 2$$

（假设信号是条件非平稳的，减 2 是为了使高斯噪声的 $\mathcal{K} = 0$）

**实现步骤**：

1. 用短时傅里叶变换（STFT）或 1/3 倍频程滤波器组分解信号到不同频带
2. 计算每个频带的时间序列的峭度
3. 在频率-带宽平面上绘制 Kurtogram
4. 选择峭度最大的频带
5. 在该频带做带通滤波 → 包络解调 → 识别故障频率

**文献中的优点**（引自 [Antoni & Randall, 2006]）：

- 实现简单，适合早期故障筛查
- 自动定位频带，不需要人工选择

**文献中的局限**（引自 [Antoni & Randall, 2006; Smith et al., 2019]）：

- 强噪声会压缩谱峭度值，使其区分度降低
- 非故障的随机冲击也会产生高峭度，导致频带误选
- 不宜作为最终判决层，适合做候选频带生成器

**适用问题**：Q1（辅助频带定位，与包络谱结合使用）

### 4. 循环谱密度分析（Cyclostationary Analysis）

**类别**：频域 + 循环平稳统计

**核心原理**：

- 故障齿轮/轴承产生**二阶循环平稳**信号：其自相关函数是时间的周期函数
- 循环谱密度（Cyclic Spectral Density, CSD）是自相关函数对时间和延迟的二维 Fourier 变换
- 在**循环频率** $\alpha = f_0, 2f_0, \ldots$ 处出现非零循环谱，指示故障频率
- $\alpha = 0$ 退化为传统功率谱；$\alpha \neq 0$ 的切片被称为**循环谱**

**关键公式**：

循环自相关函数：

$$R_x(\tau, \alpha) = \langle x(t + \tau/2) x^*(t - \tau/2) e^{-j2\pi\alpha t} \rangle_t$$

循环谱密度：

$$S_x^\alpha(f) = \int R_x(\tau, \alpha) e^{-j2\pi f \tau} \, d\tau$$

**实现步骤**（以 FFT 累积法为例）：

1. 将长序列分成多个重叠的时间块
2. 每块做 FFT，得到复谱
3. 对每对频率 $(f + \alpha/2, f - \alpha/2)$，计算谱相关
4. 对所有时间块平均
5. 在 $\alpha$ 轴上搜索峰值，对应故障频率

或使用简化版：**循环调制谱（CMS）**，计算量更小。

**文献中的优点**（引自 [Randall et al., 2001; Buzzoni et al., 2018]）：

- 理论严密，可从循环频率 $\alpha$ 直接读取故障特征频率
- 平稳噪声在 $\alpha \neq 0$ 处没有循环平稳性，天然具有抗噪性
- 不同故障源的循环频率不同，可在 $\alpha$ 域分离

**文献中的局限**：

- 计算量显著大于功率谱和包络谱
- 需要较长的信号以可靠估计循环统计量
- 论文中实现复杂度较高（可简化）

**适用问题**：Q1（频率检测）、Q3（$\alpha$ 域天然支持多源分离）

## 第二部分：统计检测与参数估计模型

### 5. 广义似然比检验（GLRT）

**类别**：统计假设检验

**核心原理**：

- 将故障检测写成两个统计假设之间的选择问题
- $H_0$：只有噪声（无故障）
- $H_1$：存在正弦信号 + 噪声（有故障）
- 对每个候选频率计算似然比统计量 $T(f)$
- 若 $\max_f T(f) > \eta$（门限），则判定有故障
- 门限可通过理论分布（卡方近似）或 Monte Carlo 仿真确定

**关键公式**：

$$H_0: x(t) = n(t)$$
$$H_1: x(t) = a\sin(2\pi ft) + b\cos(2\pi ft) + n(t)$$

对候选频率 $f$，设计矩阵 $X_f$（列向量为 $\sin(2\pi f t_n)$ 和 $\cos(2\pi f t_n)$），投影矩阵：

$$P_f = X_f (X_f^T X_f)^{-1} X_f^T$$

检测统计量：

$$T(f) = \frac{\mathbf{x}^T P_f \mathbf{x}}{\hat{\sigma}^2}$$

如果 $\max_f T(f) > \eta$，则拒绝 $H_0$（判定存在故障）。

**门限确定方法**：

1. 卡方近似：$T(f)$ 在 $H_0$ 下近似 $\chi^2_2$ 分布
2. Monte Carlo：生成纯噪声数据，计算 $\max_f T(f)$ 的经验分布，取 $(1-\alpha)$ 分位数
3. 需考虑频率扫描带来的**多重检验**问题（Bonferroni 校正或极值分布）

**实现步骤**：

1. 构造候选频率集合（可以基于功率谱粗定位或等间距扫描）
2. 对每个候选频率 $f$：
   - 构造设计矩阵 $X_f$
   - 计算投影 $P_f$ 和统计量 $T(f)$
3. 求 $\max_f T(f)$
4. 与门限 $\eta$ 比较，做出判决
5. 若判定有故障，以 $\arg\max_f T(f)$ 为频率估计

**文献中的优点**：

- 高斯白噪声假设下有明确的统计理论基础
- 可报告检测概率 $P_D$ 和误报概率 $P_{FA}$
- 比纯视觉判断频谱峰更严格

**文献中的局限**：

- 假设噪声为高斯白噪声（实际噪声若有色或有脉冲异常需先预白化）
- 大范围频率扫描时多重检验问题显著
- 频率扫描量大时计算负担较重

**适用问题**：Q1（故障存在性判决 + 频率估计）、Q4（$P_D$, $P_{FA}$ 直接作为布局优化目标）

### 6. 匹配滤波

**类别**：经典信号检测

**核心原理**：

- 对已知信号形式（正弦波）、未知参数（$A, f, \varphi$）的检测问题
- 匹配滤波器是使输出 SNR 最大化的线性滤波器
- 对于正弦信号，匹配滤波等价于对信号做 Fourier 变换（在频率域做内积）
- 与 GLRT 共享数学基础：匹配滤波 + 包络检波 + 门限判决

**关键公式**：

匹配滤波输出（对候选频率 $f$）：

$$y(f, t) = x(t) * h_f(t)$$

其中 $h_f(t) = \sin(2\pi f(T-t))$ 为匹配滤波器的冲激响应（时间反转的正弦波）

等效于：

$$Y(f) = \left| \int_0^T x(t) e^{-j2\pi f t} \, dt \right|$$

**实现步骤**：

1. 将信号与候选频率的正弦/余弦模板做相关（即计算 DFT）
2. 取模值作为检测统计量
3. 与门限比较
4. 门限可通过 Neyman-Pearson 准则确定

**文献中的优点**：

- 白高斯噪声下的**最优线性检测器**（最大化 SNR）
- 概念简单，与 FFT 有直接对应关系

**文献中的局限**：

- 需要已知或可扫描信号频率
- 信号形式必须已知（本题是正弦波，匹配良好）
- 效果等价于周期图 + 门限，优势在于统计解释更完善

**适用问题**：Q1（与 GLRT 互为等价表述）

## 第三部分：多源分离与稀疏模型

### 7. LASSO（L1 正则化稀疏回归）

**类别**：稀疏重构 / 压缩感知

**核心原理**：

- 在细网格频率点上构造正弦-余弦字典 $D$
- 假设故障频率很少（稀疏性），在普通最小二乘损失基础上加 L1 惩罚项
- L1 惩罚会**自动将不重要频率的系数压缩为零**，仅保留少数真正故障频率

**关键公式**：

字典矩阵：

$$D = [\sin(2\pi f_1 t), \cos(2\pi f_1 t), \ldots, \sin(2\pi f_M t), \cos(2\pi f_M t)]$$

LASSO 优化目标：

$$\min_{\boldsymbol{\theta}} \frac{1}{2} \| \mathbf{x} - D\boldsymbol{\theta} \|_2^2 + \lambda \|\boldsymbol{\theta}\|_1$$

参数 $\lambda$ 控制稀疏程度：$\lambda$ 越大，保留的频率越少。

**$\lambda$ 选择方法**：

1. **交叉验证**：分割数据，选使预测误差最小的 $\lambda$
2. **BIC**（贝叶斯信息准则）：$\text{BIC} = N\ln(\hat{\sigma}^2) + K\ln(N)$，选使 BIC 最小的模型
3. **噪声方差法**：当残差方差接近已知噪声方差时停止

**实现步骤**：

1. 构造频率字典 $D$（超完备基，$M \gg K$）
2. 用坐标下降法（Coordinate Descent）或 ADMM 求解 LASSO
3. 提取非零系数对应的频率 $\{f_k\}$
4. 以 LASSO 结果为初值，做联合非线性最小二乘精修

**文献中的优点**：

- 不需要预先知道故障源数量 $K$
- 全局凸优化，有唯一解
- 建模过程完整（字典构建 → 稀疏求解 → 精修）

**文献中的局限**：

- 网格过粗损失精度；网格过密导致相邻字典列高度相关（违反 RIP 条件）
- $\lambda$ 的选择对结果敏感
- 两个故障频率很近时可能只选出一个

**适用问题**：Q3（自动识别多个故障频率及其振幅和相位）

### 8. OMP（正交匹配追踪）

**类别**：贪婪稀疏重构

**核心原理**：

- 逐步添加与当前残差相关性最大的字典原子（频率分量）
- 每次添加后，对已选频率集合重新做联合最小二乘拟合
- 通过 BIC 或残差改进幅度决定停止条件

**实现步骤**：

1. 初始化：残差 $\mathbf{r}_0 = \mathbf{x}$，活跃集 $S_0 = \emptyset$
2. 计算所有候选频率与当前残差的相关性
3. 选出相关性最大者加入活跃集 $S$
4. 对 $S$ 中频率做联合最小二乘：$\min \| \mathbf{x} - D_S \boldsymbol{\theta}_S \|_2^2$
5. 更新残差：$\mathbf{r} = \mathbf{x} - D_S \hat{\boldsymbol{\theta}}_S$
6. 检查停止条件（BIC 不再下降 / 残差改善不足 / 新分量未通过 GLRT 门限）
7. 若未停止，返回步骤 2

**停止条件**（可选）：

- BIC 上升
- 残差方差下降 $< 1\%$
- 新频率的振幅估计小于噪声水平

**文献中的优点**：

- 计算速度通常快于 LASSO
- 天然输出频率数量，不需要额外做模型选择
- "OMP 粗选 + 联合非线性精修"是文献推荐的标准路线

**文献中的局限**：

- 贪心策略不保证全局最优
- 近频时 OMP 可能只选一个，漏掉另一个
- 对噪声水平敏感

**适用问题**：Q3（自动识别多个故障频率及参数）

## 第四部分：高分辨率频率估计

### 9. MUSIC（多重信号分类）

**类别**：子空间方法

**核心原理**：

- 将时间序列排列成延迟向量（Hankel 结构），估计协方差矩阵
- 特征值分解后，大特征值对应信号子空间，小特征值对应噪声子空间
- 两个子空间正交。对任意候选频率 $f$，构造导向向量 $\mathbf{a}(f)$
- 若 $f$ 是真实信号频率，$\mathbf{a}(f)$ 在信号子空间内，与噪声子空间正交 → 伪谱峰值 → $\infty$
- MUSIC 伪谱可以实现**超分辨率**（超越 FFT 的 $\Delta f$ 限制）

**关键公式**：

延迟向量：

$$\mathbf{y}_n = [x_n, x_{n+1}, \ldots, x_{n+L-1}]^T$$

协方差矩阵：

$$R = \frac{1}{Q} \sum_n \mathbf{y}_n \mathbf{y}_n^T$$

特征值分解：

$$R = E_S \Lambda_S E_S^H + E_N \Lambda_N E_N^H$$

MUSIC 伪谱：

$$P_{\text{MUSIC}}(f) = \frac{1}{\mathbf{a}(f)^H E_N E_N^H \mathbf{a}(f)}$$

其中 $\mathbf{a}(f) = [1, e^{j2\pi f/f_s}, e^{j2\pi f \cdot 2/f_s}, \ldots]^T$ 为导向向量。

**源数估计**（确定 $K$）：

- **MDL**（最小描述长度）：$\text{MDL}(k) = -2\ln L(\hat{\Theta}_k) + k\ln(Q)$
- **AIC**（赤池信息准则）：$\text{AIC}(k) = -2\ln L(\hat{\Theta}_k) + 2k$
- **特征值间隙**：找特征值突然变小的位置

**实现步骤**：

1. 选延迟长度 $L$（建议 $L \approx N/3$ 到 $N/2$）
2. 构造延迟向量并估计协方差矩阵 $R$
3. 对 $R$ 做特征值分解
4. 用 MDL/AIC 估计源数 $K$
5. 取 $K$ 个最大特征值对应的向量为 $E_S$，其余为 $E_N$
6. 在频率范围计算 MUSIC 伪谱，取前 $K$ 个峰
7. 用谐波回归精修频率并估计振幅和相位

**文献中的优点**：

- 在窄带正弦模型成立时，频率分辨率**超越** FFT 极限
- 适合研究两个相近频率的辨识问题（Q3 的近频实验）
- 伪谱图直观，适合论文可视化

**文献中的局限**：

- 需要选择延迟长度 $L$（影响协方差估计质量）
- 需要估计源数 $K$（MDL/AIC 在低 SNR 下可能不准）
- MUSIC 只输出频率，振幅和相位需要另外估计
- 对噪声模型敏感（假设白噪声）

**适用问题**：Q3（近频辨识极限实验的高分辨率对照方法）

---

### 10. ESPRIT（通过旋转不变性估计信号参数）

**类别**：子空间方法（MUSIC 的变体）

**核心原理**：

- 与 MUSIC 共享子空间分解框架
- 利用信号子空间的**旋转不变性**（延迟向量的平移结构），通过求解矩阵特征值直接得到频率估计
- **不需要搜索伪谱**，给出闭式频率估计

**关键公式**：

将协方差矩阵的信号子空间 $E_S$ 分成两个重叠的子矩阵：

$$E_S^{(1)} = [I_{L-1}, \mathbf{0}] E_S, \quad E_S^{(2)} = [\mathbf{0}, I_{L-1}] E_S$$

（去掉最后一行 / 去掉第一行）

则有：$E_S^{(2)} = E_S^{(1)} \Phi$

频率估计：

$$f_k = \frac{f_s}{2\pi} \arg(\lambda_k)$$

其中 $\lambda_k$ 为 $\Phi$ 的特征值。

**实现步骤**：

1. 同 MUSIC：构造延迟向量，估计协方差矩阵，特征值分解
2. 提取信号子空间 $E_S$
3. 构造 $E_S^{(1)}$ 和 $E_S^{(2)}$
4. 求解 $\Phi = (E_S^{(1)})^+ E_S^{(2)}$（最小二乘或总体最小二乘 TLS）
5. 计算 $\Phi$ 的特征值 $\lambda_k$，由 $\arg(\lambda_k)$ 得到频率估计

**文献中的优点**：

- 计算快（不需要频率扫描搜索伪谱）
- 频率估计精度通常高于 MUSIC
- 有闭式解

**文献中的局限**：

- 依赖延迟向量的平移不变结构（本题满足）
- 同样需要估计源数和选择延迟长度
- 低 SNR 下稳定性不如 MUSIC（可用 TLS-ESPRIT 改善）

**适用问题**：Q3（近频辨识的实用工具，替代或补充 MUSIC）

## 第五部分：信号分解与去噪模型

### 11. 奇异谱分析（SSA）

**类别**：无参数信号分解

**核心原理**：

- 将一维时间序列嵌入为二维轨迹矩阵（Hankel 矩阵）
- 对轨迹矩阵做 SVD，得到奇异值和左右奇异向量
- 稳定正弦波对应两个相近的奇异值（信号子空间的一对正交基）
- 噪声能量分散在小奇异值中
- 保留前几个奇异分量，对角平均重构，得到去噪序列

**关键公式**：

轨迹矩阵（窗口长度 $L$）：

$$X = \begin{bmatrix}
x_1 & x_2 & \cdots & x_{N-L+1} \\
x_2 & x_3 & \cdots & x_{N-L+2} \\
\vdots & \vdots & \ddots & \vdots \\
x_L & x_{L+1} & \cdots & x_N
\end{bmatrix}$$

SVD：$X = U \Sigma V^T = \sum_{i=1}^{L} \sigma_i \mathbf{u}_i \mathbf{v}_i^T$

分量分组：保留前 $r$ 个三元组 $(\sigma_i, \mathbf{u}_i, \mathbf{v}_i)$，其余置零

对角平均：从重构矩阵还原为一维序列

**实现步骤**：

1. 选择窗口长度 $L$（一般 $L < N/2$，且 $L$ 应为故障周期 $f_s/f_0$ 的整数倍效果更好）
2. 构造 Hankel 矩阵
3. SVD 分解
4. 观察奇异值谱，选择保留阶数 $r$（通常取前 $1 \sim 3$ 对分量）
5. 对角平均重构 → 去噪信号
6. 对去噪信号做 FFT / 包络谱分析或直接拟合正弦波

**文献中的优点**：

- 不需要假设信号模型（不要求是正弦波）
- 对有色噪声和缓慢变化分量有一定适应性
- 能直接展示去噪效果（时域前后对比）

**文献中的局限**：

- 窗口长度 $L$ 和保留阶数 $r$ 具有主观性
- 对角平均过程可能轻微改变振幅和相位
- 纯正弦信号在 SSA 中被分解为两个分量（一对奇异值），需要正确配对

**适用问题**：Q2（波形恢复的对照方法）

### 12. 变分模态分解（VMD）

**类别**：自适应信号分解

**核心原理**：

- 将信号分解为 $K$ 个有限带宽的**本征模态函数（IMF）**，每个 IMF 围绕一个中心频率
- 通过变分优化：最小化所有 IMF 的带宽之和，同时约束各 IMF 之和等于原始信号
- 与 EMD（经验模态分解）不同，VMD 有严格的数学优化框架

**关键公式**：

$$\min_{\{u_k\}, \{\omega_k\}} \sum_k \left\| \partial_t \left[ \left( \delta(t) + \frac{j}{\pi t} \right) * u_k(t) \right] e^{-j\omega_k t} \right\|_2^2$$

$$\text{s.t.} \quad \sum_k u_k(t) = x(t)$$

其中 $u_k$ 为第 $k$ 个模态，$\omega_k$ 为其中心频率。

**实现步骤**：

1. 选择分解模态数 $K$
2. 初始化 $\{u_k\}$ 和 $\{\omega_k\}$ 的频域表示
3. 交替方向乘子法（ADMM）迭代更新：
   - 更新每个模态的频域表示（Wiener 滤波）
   - 更新中心频率（模态功率谱重心）
   - 更新 Lagrange 乘子
4. 直到收敛
5. 对每个模态做频谱分析，识别含故障信息的模态

**文献中的优点**：

- 有数学优化理论基础，比 EMD 更稳健
- 可以将噪声和信号分离到不同模态中
- 对多源故障，不同故障频率可能分离到不同模态

**文献中的局限**：

- 需要预设模态数 $K$（可结合频谱峰值个数估计）
- 计算量大于 SSA
- 模态混叠问题仍然存在

**适用问题**：Q2（波形恢复对照）、Q3（多源分离的分解式方案）

---

### 13. 自适应滤波 + 局部均值分解（LMD）

**类别**：自适应滤波 + 时频分解

**核心原理**（引自 [Darong et al., 2018]）：

- 先用自适应滤波（如改进的 RLS 算法）压制强背景噪声
- 再用 LMD 将信号分解为若干乘积函数（PF）分量
- LMD 是 EMD 的改进版，通过滑动平均求局部均值函数和包络函数

**实现步骤**：

1. RLS 自适应滤波：以延迟信号为参考，自适应调整滤波器系数压制宽带噪声
2. LMD 分解：
   - 求信号的局部极值点
   - 滑动平均得到局部均值函数和包络函数
   - 从信号中减去局部均值，除以包络（解调）
   - 迭代直到得到纯调频信号
   - 将包络函数与纯调频信号相乘得到 PF 分量
3. 对各 PF 分量做频谱分析或包络谱分析

**文献中的优点**：

- 对强随机噪声场景实用
- LMD 可处理非平稳信号

**文献中的局限**：

- 物理可解释性弱于循环平稳路线 [Buzzoni et al., 2018]
- 自适应滤波需要选择阶数和遗忘因子
- LMD 的端点效应和模式混叠

**适用问题**：Q2（信号去噪与恢复的对照方法）

## 第六部分：解卷积增强模型

### 14. MCKD（最大相关峭度解卷积）

**类别**：盲解卷积

**核心原理**（引自 [McDonald et al., 2012]）：

- 设计一个 FIR 滤波器，使滤波后信号的**相关峭度**最大化
- 相关峭度与传统峭度不同：不仅惩罚冲击性，还要求冲击以已知的故障周期 $T$ 重复出现
- 通过显式利用故障周期，增强周期性冲击，同时压制随机冲击和噪声

**关键公式**：

相关峭度（Correlated Kurtosis）：

$$CK_M(T) = \frac{\sum_{n=1}^{N} \left( \prod_{m=0}^{M} y_{n-mT} \right)^2}{\left( \sum_{n=1}^{N} y_n^2 \right)^{M+1}}$$

其中 $y_n$ 是滤波输出，$T = f_s / f_0$ 是故障周期，$M$ 是相关偏移数（一般 $M = 1$ 到 $7$）。

滤波器系数通过迭代求解得到。

**实现步骤**：

1. 估计或已知故障周期 $T$（可从转速 / 齿轮参数得到，或从功率谱粗估）
2. 选择滤波器长度 $L_f$ 和 $M$
3. 迭代计算滤波器系数使 $CK_M(T)$ 最大化
4. 对滤波后信号做包络谱分析
5. 在包络谱中识别故障频率及其谐波

**文献中的优点**：

- 对低 SNR 下的周期冲击信号效果显著
- 比传统峭度解卷积（MED）更稳健
- 可与 Kurtogram + 包络谱形成链式检测方案

**文献中的局限**：

- 需要故障周期 $T$ 的先验知识
- 对转速波动敏感（$T$ 不固定时效果下降）
- 滤波器长度和 $M$ 需要调节

**适用问题**：Q1（前端信号增强）、Q2（增强后波形更清晰）、Q3（结合 OMP 做多源增强）

---

### 15. 循环平稳盲解卷积（CYCBD）

**类别**：盲解卷积

**核心原理**（引自 [Buzzoni et al., 2018]）：

- 寻找一个滤波器，使输出信号的**循环平稳性**最大化
- 与 MCKD 不同：不是最大化冲击峭度，而是最大化循环频率 $\alpha$（故障频率）处的循环谱能量
- 更贴近故障形成机理：故障信号本质是循环平稳的，而不是单纯脉冲性的

**关键公式**：

<!-- 循环平稳指标（在循环频率 $\alpha$ 处）：

$$J(\mathbf{h}) = \frac{\sum_f |S_y^\alpha(f)|^2}{\sum_f |S_y^0(f)|^2}$$

其中 $S_y^\alpha(f)$ 是滤波输出的循环谱密度。最大化 $J(\mathbf{h})$ 得到滤波器系数。 -->

**实现步骤**：

1. 估计或已知故障特征频率 $f_0$（设 $\alpha = f_0$）
2. 选择滤波器长度
3. 迭代求解使循环平稳指标最大化的滤波器系数
4. 对滤波后信号做包络谱或循环谱分析

**文献中的优点**：

- 比 MED（最大峭度）和 MCKD 对随机冲击干扰更稳健
- 直接针对故障信号的物理本质（循环平稳）
- 适合复杂传递路径场景

**文献中的局限**：

- 计算量大于 MCKD
- 仍需故障频率先验
- 实现复杂度较高

**适用问题**：Q1（前端增强的高阶方案）、Q3（多源的循环平稳分离）

## 第七部分：传感器布局优化模型

### 16. 鲁棒传感器布局优化

**类别**：约束组合优化

**核心原理**：

- 设备上有 $M$ 个候选测点，从中选 3 个放置传感器
- 每个测点到每个故障源的传播增益不同、噪声方差不同
- 目标：最大化系统对**最难检测的故障源**的检测概率（短板效应）
- 约束：总误报率不超过 $\alpha$；3 个传感器不能过于集中

**决策变量**：

$$z_j = \begin{cases} 1, & \text{选择位置 } j \\ 0, & \text{不选择} \end{cases}$$

**约束**：

$$\sum_{j=1}^{M} z_j = 3$$

**目标函数**：

$$\max_{\mathbf{z}} \left\{ \min_k P_{D,k}(\mathbf{z}) - \lambda R(\mathbf{z}) \right\}$$

其中：

- $P_{D,k}(\mathbf{z})$：布局 $\mathbf{z}$ 下对故障源 $k$ 的检测概率
- $R(\mathbf{z})$：惩罚项（传感器距离过近或噪声相关性过高）
- $\text{s.t.} \quad P_{FA} \leq \alpha$

**关键参数**：

- 传播增益 $g_{jk}$：位置 $j$ 对故障源 $k$ 的衰减系数
- 噪声方差 $\sigma_j^2$：位置 $j$ 处的噪声水平
- 等效信噪比：$\text{SNR}_{jk} = g_{jk}^2 A_k^2 / \sigma_j^2$

**求解方法**：

1. 候选位置少（$\leq 15$）：枚举所有 $C_M^3$ 组合
2. 候选位置多：遗传算法（GA）或模拟退火
3. 对每种布局做 Monte Carlo 仿真，估计 $P_D$ 和 $P_{FA}$
4. 比较单传感器、随机三点、优化三点的性能

**验证方案**：

- 构造仿真场景（定义几何布局、传播函数、噪声场）
- 改变全局 SNR，绘制 $P_D$ vs SNR 曲线
- 敏感性分析：传播模型参数变化时布局的稳健性

**说明**：题目未提供设备几何和测点数据，此模型为**通用框架**，需要自行设计仿真场景。论文需明确区分基于真实数据的结论和基于仿真布局的结果。

**适用问题**：Q4（传感器布局优化）

## 实验对比设计建议

用以下同一套指标和同一组合成数据测试每个模型：

### 合成数据生成

```python
# 单源故障
fs = 100
T = 400
t = np.arange(0, T, 1/fs)
A = 0.035
f0 = 2.0
phi = np.random.uniform(0, 2*np.pi)
noise = np.random.randn(len(t)) * sigma  # sigma 使 SNR ≈ -12 dB
x = A * np.sin(2*np.pi*f0*t + phi) + noise
```

### 统一评测指标

| 指标 | 含义 | 计算方式 |
|------|------|---------|
| $\Delta f$ | 频率估计误差 | $\| \hat{f} - f_0 \|$ |
| $\Delta A / A$ | 振幅相对误差 | $\| \hat{A} - A \| / A$ |
| $\Delta \varphi$ | 相位误差 | $\| \hat{\varphi} - \varphi_0 \|$（模 $2\pi$）|
| $P_D$ | 检测概率 | 1000 次独立实验中正确检测的次数 / 1000 |
| $P_{FA}$ | 误报率 | 纯噪声数据中误判有故障的比例 |
| RMSE | 波形恢复误差 | $\sqrt{\frac{1}{N}\sum (\hat{s}_n - s_n)^2}$ |

### 对比实验矩阵

| 实验 | SNR 范围 | 测量量 |
|------|----------|--------|
| 频率估计精度 vs SNR | -20 dB → -5 dB | $\Delta f$ |
| 检测概率 vs SNR | -20 dB → -5 dB | $P_D$（固定 $P_{FA}=5\%$）|
| 振幅/相位精度 vs SNR | -20 dB → -5 dB | $\Delta A/A, \Delta\varphi$ |
| 近频辨识极限（Q3） | 固定 SNR | 两个频率正确分辨的最小 $\Delta f$ |
| 计算时间 | — | 运行时间（秒）|
