\noindent{\zihao{-4}\bfseries\songti 摘 要：}

机械设备故障产生的周期信号能量微弱，常被电机运转和传动系统产生的强噪声完全淹没。传统谱峭度法在强噪声下峭度被压缩、易导致误报；最大相关峭度解卷积则依赖已知故障周期——而本题中故障频率恰好是待求的未知量。本文以\textbf{广义似然比检验（GLRT）}为核心框架，构建了从单频检测、波形恢复、多分量分离到传感器布局优化的渐进式方法链。

    \textbf{针对问题一：}对\texttt{data.xlsx}中"单源故障"工作表的数据进行线性去趋势后，采用归一化周期图扫描与纯噪声\textbf{Monte Carlo 门限校准}，在目标虚警率约$5\%$下检出故障频率$f_0 = 2.0000015$ Hz。相较于传统FFT峰值法，GLRT的核心优势在于提供了具有明确概率意义的统计判决门限，并通过连续频率精修突破FFT栅格分辨率限制。

    \textbf{针对问题二：}以问题一频率为初值建立\textbf{谐波回归}模型，恢复出$\hat{s}(t) = 0.03519 \sin(2\pi \times 2.0000016t + 1.5736)$，并采用局部渐近正态近似量化了参数的不确定性。\textbf{奇异谱分析（SSA）}对照波形的相关系数达$0.9956$；合成数据实验覆盖SNR $-20$至$-5$ dB，参数区间覆盖率均超过$93.5\%$。

    \textbf{针对问题三：}将单频GLRT扩展为基于\textbf{逐次干扰消除（SIC）}的顺序检测器，以联合谐波回归和\textbf{BIC准则}自动确定分量数量。在"多源故障"工作表中自动识别出4个周期分量，频率分别约为4、8、13、14 Hz。SNR $= -12$ dB时正确定阶率为$98.0\%$；条件化近频实验表明，$T = 400$ s条件下90\%经验辨识间隔约0.002--0.003 Hz，随振幅比增大而增宽。

    \textbf{针对问题四：}采用V1参数化结构模型，在6个区域生成120个无量纲候选点，经预筛选和组合搜索后以Monte Carlo验证排序。最优布局为齿轮箱右区点\texttt{v1\_p105}与齿轮箱左区点\texttt{v1\_p074}、\texttt{v1\_p062}，三点通过互补频响覆盖全部4个目标频率。SNR $= -12$ dB时平均分量检出率为$84.25\%$，误报率为$3.67\%$。该结论适用于本文给定的参数化几何与传播响应设置，不能直接解释为真实设备上的物理最优安装坐标。

    最后，本文方法链以GLRT为统一框架，具有虚警率可经验校准、分量数量可自动确定、辨识界限可条件化刻画的特点。四问从检测到部署共享核心统计框架，形成了完整的微弱故障信号处理方案。

\section{问题重述}
\subsection{问题背景}
机械设备长期运行后可能出现齿轮破损、轴承磨损等故障。故障引起的周期振动幅值较小，常被电机、传动系统和环境噪声掩盖，因此需要同时控制漏检和误报。

从故障诊断领域的研究共识来看，强噪声下微弱故障检测的关键在于围绕故障的\textbf{周期结构}而非单纯的脉冲强度来设计检测链路\upcite{randall2001relationship}。传统工业诊断方法如谱峭度（Kurtogram）通过寻找最大冲击频带来定位故障\upcite{antoni2006spectral}，但强噪声会压缩谱峭度，非故障冲击也会产生高峭度，导致误选频带和误报。最大相关峭度解卷积（MCKD）利用故障周期先验增强微弱周期冲击\upcite{mcdonald2012mckd}，在低信噪比下效果优于传统最小熵解卷积，但依赖已知故障周期。针对性频带选择方法进一步表明，频带选择须围绕已知故障频率才能有效避免无关冲击的干扰\upcite{smith2019optimal}。这些研究共同指向：\textbf{能否显式利用故障信号的周期性先验，是提高检出率并压低误报率的关键}。

\subsection{问题要求}
假设某机械可能存在故障齿轮，其产生的微弱周期信号模型$s(t)$为：
\begin{equation}
s (t) = A \sin (2 \pi f _ {0} t + \phi_ {0}),
\end{equation}
其中$A$为信号振幅，$f_{0}$为故障特征频率，$\phi_{0}$为初相（三者均未知）。采集到的信号为$x(t) = s(t) + n(t)$。

基于附件data.xlsx，建立数学模型解决以下问题：
\begin{itemize}
    \item [问题一] 建立一个能够有效检测并识别出淹没在强噪声中的微弱周期信号特征的数学模型。模型应能准确求解出目标频率$f_{0}$的具体数值，并说明该方法相较于传统傅里叶变换的优势。
    \item [问题二] 在成功检测出$f_{0}$的基础上，进一步建立数学模型，实现对原始微弱周期信号$s(t)$的波形提取与恢复。给出恢复后的信号表达式，并从时域和频域两个角度展示恢复效果，定量评价恢复误差。
    \item [问题三] 在实际工况中，机械设备可能同时存在多处不同位置的故障，导致采集到的信号中包含多个不同频率的微弱周期信号分量：$x(t) = \sum_{i=1}^{M} A_i \sin(2\pi f_i t + \phi_i) + n(t)$。请基于前两问建立的模型，提出一种扩展方案使其能够从单一通道的混合信号中自动分离并识别出多个潜在的周期分量频率$f_1, f_2, \cdots, f_M$，并估计对应的振幅$A_i$和初相$\phi_i$。设计数值实验验证方法，讨论当两个频率非常接近时的辨识极限。
    \item [问题四] 假设可以在设备的关键部位布置最多3个加速度传感器。考虑到信号传播路径的衰减、不同测点对同一故障源的敏感度差异以及噪声的空间分布不均，请建立一个数学模型，为这3个传感器设计最优的布局方案。目标是最大化整个检测系统对微弱故障信号的综合检测概率，并最小化误报率。方案需结合前三问的结论，解释布局策略为何能提升系统的鲁棒性和可靠性，并设计实验进行仿真验证。
\end{itemize}

\section{问题分析}
\subsection{问题一的分析}
问题一要求在强噪声背景下检测微弱周期信号的频率$f_0$。核心难点在于：(1) 信号能量远低于噪声能量（估计SNR为$-12.08$ dB，功率比约16倍，幅度比约4倍）；(2) 故障频率$f_0$、振幅$A$和初相$\phi_0$均未知；(3) 传统方法各有局限。

FFT功率谱法最为简单但缺乏显式的统计判决阈值——无法区分"微弱信号存在"与"纯噪声偶然波动"。工业界常用的谱峭度法（Kurtogram）\upcite{antoni2006spectral}在强噪声下峭度被压缩，且非故障冲击同样产生高峭度，误报风险高。MCKD\upcite{mcdonald2012mckd}和循环平稳盲解卷积\upcite{buzzoni2018blind}需要故障周期作为先验——这在本题中恰好是待求的未知量。

广义似然比检验（GLRT）\upcite{kay1998fundamentals}为本题提供了理想框架。GLRT在FFT栅格频率上的计算可退化为归一化周期图的最大值检验；其独特贡献在于：(1) 有基于蒙特卡洛仿真的统计判决门限，可控制多频点联合扫描的虚警率；(2) 可自然扩展到未知参数的联合估计；(3) 增加了连续频率精修步骤。由于GLRT的信号模型（正弦+高斯噪声）与题设完全匹配，在$N$充分大时该检验是一致的\upcite{kay1998fundamentals}。

\subsection{问题二的分析}
问题二需要在已知$f_0$的基础上恢复$s(t)$的振幅$A$和初相$\phi_0$。核心难点在于：(1) 频率、振幅、初相耦合——Q1精修频率的统计不确定度（约$10^{-5}$ Hz）经$N=40001$放大后，若直接固定频率，振幅和相位估计会产生偏移；(2) 真实数据无波形真值，验证只能依靠残差诊断、谱峰抑制和多方法交叉对照间接完成；(3) 点估计不足以评价可靠性，Bootstrap对$N=40001$的非线性回归计算量过高。

直接最小二乘拟合给出点估计但不提供不确定性，也未检验模型充分性。窄带滤波\upcite{smith2019optimal}可提取目标频带波形，但滤波器参数缺乏统计最优准则。SSA\upcite{golyandina2001ssa}能自适应分解信号，但分量选取依赖人工且对窗口长度敏感。

本文以Q1精修频率为初值，对频率和线性参数联合非线性最小二乘，基于局部渐近正态近似以$10000$次多元正态抽样给出95\%参数区间。验证分三层：残差GLRT重扫描检验遗漏分量；BIC比较公共/独立频率模型检验频率恒定性；SSA与窄带滤波从不同原理恢复波形交叉验证。合成数据在$-20$至$-5$ dB共5个SNR水平上定量评估波形RMSE和区间覆盖率。

\subsection{问题三的分析}
问题三将场景从单周期分量扩展到多周期分量。核心挑战在于：(1) 分量数量$M$未知——过估计产生伪分量，欠估计遗漏真实故障；(2) 两频率接近时（$\Delta f$在$10^{-3}$ Hz量级），正弦/余弦基向量高度相关，设计矩阵病态，最小二乘解方差急剧放大；(3) 顺序检测的虚警率会累积，须在控制整体虚警率的前提下检测多个微弱分量；(4) 单通道数据只能识别显著周期分量，缺少转速、齿数等机械先验时，不能将分量唯一映射到独立物理故障源。

MUSIC算法\upcite{schmidt1986music}频率分辨率高于周期图但需预知$M$，低SNR下伪谱峰展宽严重。ICA假设源信号非高斯，与正弦+高斯噪声模型不匹配。EMD缺乏频率选择性，近频正弦极易模态混叠。LASSO可在频率字典上一次选择所有分量，但正则化参数对结果敏感，近频时字典基强相关性破坏稀疏恢复条件。

本文借鉴SIC思想：Q1的GLRT扩展为多峰顺序检测器，每步虚警率控制在$\alpha_{\rm step}=0.005$以约束累积风险；单频谐波回归扩展为多频联合拟合，以SVD替代正规方程求逆，监控设计矩阵条件数；BIC自动定阶——仅当$\Delta$BIC$\ge10$才保留候选分量。针对近频簇，先用细网格定位最强分量，剥离后扫描残差弱峰，由二源联合精修和BIC决定是否拆分为两个分量。合成数据近频扫描确定辨识极限，极端工况验证适用边界。

\subsection{问题四的分析}
问题四要求为最多3个加速度传感器设计最优布局，最大化综合检测概率并最小化误报率。核心难点在于：(1) 题目未提供设备几何、材料和传播路径参数，无法计算连续空间中的全局最优坐标，任何布局结论必须声明参数化模型范围；(2) 信号传播涉及距离衰减、方向性敏感度和模态耦合，不同测点对各频率的响应差异显著，最优布局不从直觉得出；(3) 检测概率与误报率相互制约——布点过密使响应冗余，浪费通道却不提升检测能力；(4) 仅允许3个传感器，从$L$个候选点选最优三元组的搜索空间为$O(L^3)$；(5) 传播响应和噪声存在不确定性，标称最优在增益/相位扰动和相关噪声下可能退化。

经验布点缺乏定量依据。振型驱动方法依赖有限元模型，面向模态识别而非微弱信号检测。D最优/A最优等经典准则\upcite{kammer1991sensor}以回归参数估计精度为目标，在多分量检测场景需改造——本题目标是各分量检出概率而非单一估计精度。

本文以参数化结构示例（V1模型，无量纲坐标，不代表真实安装尺寸）将连续布局问题离散化：在6个区域生成120个候选点，以复响应矩阵描述各点对4个已知分量的传播增益和相位延迟；按白化响应预筛选为40个互补低噪点；定义解析评分$J_{\rm ana}$（噪声加权非中心参数+信息矩阵特征值-冗余惩罚），贪心搜索后枚举全部$\binom{40}{3}=9880$个三元组合；对靠前布局在多个SNR下执行Monte Carlo验证，每次试验施加对数正态增益扰动（$\sigma=0.15$）和相位扰动（$\sigma=0.20$ rad），噪声用$\rho=0.30$的相关高斯模型，按$J_{\rm MC}$（平均检出率+最弱分量覆盖率-误报超额惩罚）重新排序。两阶段的意义：解析评分利用标称模型快速缩小搜索范围，Monte Carlo评分在随机扰动下检验鲁棒性。

\section{模型假设}
\begin{itemize}
    \item [(1)] 假设噪声$n(t)$为零均值、独立同分布的高斯白噪声，即$n(t) \sim \mathcal{N}(0, \sigma^2)$。
    \item [(2)] 假设故障信号为单一频率的稳定正弦波，频率$f_0$在观测区间内保持恒定。
    \item [(3)] 假设观测数据时间轴均匀采样（$f_s = 100$ Hz），不存在丢点或时基抖动。
    \item [(4)] 假设观测信号可能包含直流分量和线性趋势，但可通过线性去趋势预处理予以消除。
    \item [(5)] 多周期分量情形下，假设各分量满足线性叠加，分量之间不产生互调。
    \item [(6)] 不同测点对同一周期分量的敏感度差异可用复响应$h_{mk}$描述；Q4 V1验证中的测点噪声采用相关系数$\rho=0.30$的等相关高斯模型。
\end{itemize}

\section{问题一：故障频率检测模型的建立与求解}

\subsection{数据预处理}
原始采集信号可能包含直流偏置和传感器温漂引起的线性趋势。采用线性去趋势方法：
\begin{equation}
y(t) = x(t) - (\hat{\beta}_0 + \hat{\beta}_1 t),
\label{eq:detrend}
\end{equation}
其中$\hat{\beta}_0$和$\hat{\beta}_1$由最小二乘拟合得到。预处理后$y(t)$的均值为$-1.78 \times 10^{-19}$，标准差为$0.1031$。

\subsection{GLRT检测框架}
\subsubsection{二元假设检验}
对预处理信号$\mathbf{y}$，在候选频率$f$处构造二元假设检验\upcite{kay1998fundamentals}：
\begin{align}
H_0 &: y(t_n) = n(t_n), \quad n(t_n) \sim \mathcal{N}(0, \sigma^2), \\
H_1 &: y(t_n) = A\sin(2\pi f t_n + \phi) + n(t_n).
\end{align}

由于$H_1$下$A$和$\phi$为未知冗余参数，不存在一致最优检验。GLRT将未知参数用其最大似然估计替代后构造似然比，在高斯噪声下最大似然估计等价于最小二乘估计\upcite{poor2013detection}。

\subsubsection{线性化与投影矩阵}
利用三角恒等变换$A\sin(2\pi f t + \phi) = a\sin(2\pi f t) + b\cos(2\pi f t)$，其中$a = A\cos\phi$, $b = A\sin\phi$。对$N$个离散采样点构造$N \times 2$设计矩阵$\mathbf{X}_f$，其第$n$行为$[\sin(2\pi f t_n),\ \cos(2\pi f t_n)]$。定义投影矩阵：
\begin{equation}
\mathbf{P}_f = \mathbf{X}_f(\mathbf{X}_f^T \mathbf{X}_f)^{-1} \mathbf{X}_f^T.
\end{equation}

\subsubsection{扫描统计量}
固定候选频率时，高斯未知方差模型的似然比是投影解释能量占总能量比例的单调函数\upcite{scharf2005statistical}。程序采用与FFT栅格归一化周期图一致的统计量
\begin{equation}
T(f) = \frac{\mathbf{y}^T \mathbf{P}_f \mathbf{y}}{\hat v},
\qquad
\hat v=\frac{1}{N}\|\mathbf y\|^2 ,
\label{eq:glrt_stat}
\end{equation}
其中$\hat v$为去趋势后观测的总方差。该实现没有使用候选频率下的残差方差，因此不直接套用$2F(2,N-2)$分布；全文的检测门限均由同一定义下的纯噪声Monte Carlo最大值分布给出。分子表示$\mathbf y$在频率$f$的正弦/余弦子空间中的投影能量。

\subsubsection{基于FFT的高效计算}
取FFT栅格频率$f_k = k f_s/N$为候选集，利用$\|\mathbf{P}_{f_k} \mathbf{y}\|^2 \approx \frac{2}{N}|Y(f_k)|^2$，通过FFT将频率扫描阶段的耗时降至$O(N\log N)$：
\begin{equation}
T(f_k) \approx \frac{2|Y(f_k)|^2}{N \hat{\sigma}^2}.
\end{equation}

\subsubsection{蒙特卡洛门限确定}
在多频点（$\sim2\times10^4$个）联合扫描的检测问题中，即使单频点虚警概率很低，多重比较也会使联合虚警率显著放大。借鉴雷达检测中的恒虚警率（CFAR）概念\upcite{rohling1983cfar}，采用纯噪声蒙特卡洛仿真确定经验门限：生成$M=500$组独立标准高斯白噪声（长度$N=40001$），对每组计算$T_{\max}^{(m)} = \max_{f_k} T^{(m)}(f_k)$，取上$\alpha=0.05$分位数作为门限。最终判决：
\begin{equation}
\boxed{\text{若 } T_{\max} = \max_{f} T(f) > \eta_{0.05},\ \text{则判定故障频率显著存在。}}
\end{equation}

该门限基于500次MC的95\%经验分位数，将虚警率校准至目标约5\%水平。需注意500次MC在尾部分位数处的估计精度有限（约25个样本落在尾部区域），因此门限本身具有抽样不确定性。

\subsection{频率精修}
在粗估频率$f_{\text{coarse}}$的$\pm 0.01$ Hz邻域内，采用Brent有界标量优化\upcite{brent2013algorithms}最小化最小二乘拟合残差：
\begin{equation}
f_0 = \arg\min_{f \in [f_{\text{coarse}} - \Delta,\ f_{\text{coarse}} + \Delta]} \|\mathbf{y} - \mathbf{X}_f (\mathbf{X}_f^T\mathbf{X}_f)^{-1}\mathbf{X}_f^T\mathbf{y}\|^2,
\end{equation}
连续优化避免把频率估计限制在FFT离散栅格上，优化器输出容差约为$10^{-7}$ Hz。该数值容差不是统计估计精度；Q2区间估计显示统计不确定度约为$10^{-4}$ Hz量级。

\begin{equation}

\end{equation}

\subsection{公共频率模型验证}
Randall等\upcite{randall2001relationship}强调故障识别依赖周期结构的一致性。为检验频率是否在整段观测中保持恒定，采用BIC准则比较公共频率模型与独立频率模型。BIC的计算方式为$N\log(\text{SSE}/N) + k\log N$，其中公共频率模型$k=1+3K$（1个公共频率+每段3个线性参数），独立频率模型$k=4K$（每段独立频率+3个线性参数）。400秒划分为8个50秒段，公共频率模型BIC = $-183955.07$低于独立频率模型的$-183885.11$，表明数据与公共频率模型更一致。需要指出BIC选优不等同于严格证明了频率恒定，它只能说明在候选模型集中，公共频率模型更节俭地描述了数据；如果真实频率存在微小的时变（变化量小于各段估计的统计不确定度），BIC可能检测不到。

\section{问题三：多周期分量分离与参数识别模型的建立与求解}

\subsection{多峰顺序检测器}
将问题一的单峰GLRT扩展为基于逐次干扰消除（SIC）的多峰顺序检测器。SIC源于通信领域的信号分离思想，流程如下：

\begin{enumerate}
    \item 对预处理信号$\mathbf{y}$执行GLRT全频段扫描，取$T_{\max}=\max_f T(f)$；
    \item 若$T_{\max} > \eta_\alpha$（每步虚警率$\alpha_{\text{step}}=0.005$），将对应频率$f_k$加入候选集；
    \item 对当前候选集的全部$K$个频率进行多频联合谐波回归拟合，使用SVD分解计算最小二乘解，监控设计矩阵条件数；
    \item 计算BIC变化量：$\Delta\text{BIC} = \text{BIC}_{K-1} - \text{BIC}_K$；
    \item 若$\Delta\text{BIC} \ge 10$，保留该分量并从残差中继续搜索；否则终止并回退。
\end{enumerate}

SIC的固有局限是误差传播——早期分量的估计误差会传递给后续分量，且没有纠正早期错误的机制。针对近频簇，增加细网格扫描与二源联合非线性精修策略。

\subsection{残差诊断}
对Q3分离4个分量后的残差进行诊断。残差偏度为$-0.0143$，超额峰度为$-0.00365$；1--100阶自相关绝对值的最大值为$0.01542$（滞后21阶），100个滞后中有3个超出逐阶近似$95\%$参考界$\pm0.00980$。Ljung--Box联合检验得到$Q(100)=86.43$、$p=0.8312$，未拒绝残差无自相关的原假设；Jarque--Bera检验统计量为$1.382$、$p=0.5010$，未拒绝正态性。因而残差可近似视为高斯白噪声，未发现明显偏离模型假设的证据。需要说明的是，Q3模型仅解释了$11.92\%$的总方差；检验结果支持剩余波动主要表现为噪声，却不能证明其中绝对不存在未被正弦模型捕捉的微弱结构。

\section{问题四：传感器布局优化模型的建立与求解}

\subsection{候选测点与信号传播模型}
V1模型在轴承左区、轴承右区、齿轮箱左区、齿轮箱右区、输入轴区和输出轴区生成120个候选点。坐标为无量纲参数化结构坐标，不代表真实机器安装尺寸。候选点位置围绕各区域中心按半径$0.08,0.13,0.18,0.23$的环带生成，并加入竖向随机扰动；编号能被17整除的点除特定区域外视为不可安装。第$m$点的观测模型为

\begin{equation}
y_m(t) = \sum_{k=1}^{4} |h_{mk}| A_k \sin(2\pi f_k t + \phi_k + \arg h_{mk}) + n_m(t),
\end{equation}

其中$h_{mk}$为复响应，$n_m(t)$为测点噪声。程序使用Q3的完整估计值
\begin{align*}
\boldsymbol f&=(3.999998,7.999929,13.000089,14.000023)\ {\rm Hz},\\
\boldsymbol A&=(0.019103,0.024889,0.010503,0.040053).
\end{align*}
正文中的整数频率仅用于显示。设候选点到第$k$个参数化源的距离为$d_{mk}$、表面法向量为$\boldsymbol n_m$、传播方向为$\boldsymbol u_{mk}$，V1响应写为
\begin{align}
h_{mk}={}&\left(0.55+0.45|\boldsymbol n_m^\mathsf T\boldsymbol u_{mk}|\right)
\left[1+0.20\cos\{0.73(r_m+1)(k+1)\}\right]\notag\\
&\times\frac{\exp(-0.38d_{mk})}{d_{mk}^{0.65}}
\exp\!\left\{\mathrm i\left(1.15f_kd_{mk}+0.23r_m(k+1)\right)\right\}.
\end{align}
式中$r_m$为区域编号，上式相邻因子均为乘法。

各区域基准噪声倍率依次为$0.95,1.05,0.88,1.20,1.30,0.98$，并叠加标准差为8\%的点间扰动。程序以$|h_{mk}|/\sigma_m$构造白化响应，按最弱分量响应和平均响应预筛选，同时剔除平方余弦相似度超过0.97的冗余点，最终保留40个候选点。

\subsection{多传感器加权融合检测}
对任一三点布局$S$，每个测点计算4个目标频率上的归一化投影统计量$T_m(f_k)$。V1程序采用噪声方差倒数权重：
\begin{equation}
T_{\text{fused}}(f_k) = \sum_{m\in S} w_m T_m(f_k), \quad w_m = \frac{1/\hat{\sigma}_m^2}{\sum_{j\in S} 1/\hat{\sigma}_j^2},
\end{equation}
单点统计量已经按噪声方差归一化，因此第二次加权属于V1的工程启发式设计，不是严格联合GLRT。程序对每个布局在相同权重和噪声相关模型下重新执行300次纯噪声门限校准，以避免直接套用单点统计量分布。本文只报告该V1策略的经验性能，不声称它是所有融合权重中的理论最优解。

\subsection{布局评分函数与优化}
V1分两阶段评分。解析筛选阶段令$\tilde h_{mk}=h_{mk}/\sigma_m$，先计算噪声加权非中心参数$\lambda_k(S)$、信息矩阵$\boldsymbol G_S=\tilde{\boldsymbol H}_S^H\tilde{\boldsymbol H}_S$和响应冗余度$R(S)$。解析目标为
\begin{equation}
J_{\rm ana}(S)=\lambda_{\min}^{\rm robust}
+0.10\bar{\lambda}
+0.002\log\!\left(1+\operatorname{tr}\boldsymbol G_S\right)\bar{\lambda}
+0.002\lambda_{\min}(\boldsymbol G_S)\bar{\lambda}
-0.03R(S)\bar{\lambda},
\end{equation}
其中$\lambda_{\min}^{\rm robust}=\min_k\lambda_k(S)[1-0.20R(S)]$，$\bar\lambda$为4个分量的平均非中心参数。120点预筛选为40点后，先通过贪心搜索与单点交换缩小范围，再枚举全部$\binom{40}{3}=9880$个三点组合。解析目标最高的布局为\texttt{p105+p074+p098}，贪心加交换所得目标值比完整枚举最优值低3.34\%。

第二阶段对解析目标靠前布局和基线布局执行SNR$=-25,-20,-15,-12,-10,-5$ dB的Monte Carlo验证，每个SNR运行100次信号试验和300次纯噪声试验。传播响应增益采用对数扰动（标准差0.15），相位扰动标准差为0.20 rad，测点噪声的等相关系数为0.30。验证评分为
\begin{equation}
J_{\rm MC}(S)=0.45\bar P_D+0.35P_D^{\rm low}
+0.20\overline{\min_k P_{D,k}}
-2\max(0,\bar P_{FA}-0.05).
\label{eq:q4_score}
\end{equation}
解析目标用于缩小搜索范围，最终布局以$J_{\rm MC}$排序。

\subsection{故障分量覆盖均衡性}
V1验证评分使用各SNR下最弱分量检出率的平均值约束不均衡性。验证最优布局的该指标为$47.0\%$，固定随机三点基线为$33.67\%$。在SNR$=-12$ dB时，最弱分量检出率分别为$59.0\%$和$26.0\%$。三个最优点中，\texttt{p105}对8 Hz和14 Hz响应较强，\texttt{p074}、\texttt{p062}对13 Hz响应较强并补充4 Hz响应；互补频响而非区域数量本身决定了V1排名。

\subsection{鲁棒性验证}
本节验证未单独设置”传播阴影”等离散场景，而是在每次信号试验中统一施加随机扰动：每个测点的复响应增益乘以对数正态因子$\exp\{\mathcal N(0,0.15^2)\}$，相位增加$\mathcal N(0,0.20^2)$ rad扰动，测点噪声采用相关系数$\rho=0.30$的等相关高斯模型。表\ref{tab:q4_ranking}中的验证排名已经包含这些扰动。由于只使用单一扰动强度，结果支持的是该设定下的经验稳定性，不能外推到任意安装误差或传播阴影。

\subsection{布局策略的物理解释}
V1最优点全部位于参数化齿轮箱区域。模型把13 Hz参数化源放在齿轮箱左侧附近、14 Hz源放在齿轮箱右侧附近，且齿轮箱区域的基准噪声倍率较低；这使齿轮箱点在白化响应上占优。三个点虽属于两个区域标签，但坐标和表面法向量不同，对4个频率的复响应也不同。该结果说明在当前参数化响应矩阵中，局部频响互补带来的收益超过跨区域布点收益。真实设备是否具有相同结论，需要用有限元频响或现场移动传感器标定替换$\{h_{mk}\}$后重新优化。

\section{模型的检验}
\subsection{四问结论的内在关联}
四问构成一套渐进式方法链：Q1检测到$f_0=2.0000015$ Hz；Q2在此基础上精修参数并检验模型充分性；Q3把单分量方法扩展至$K=4$；Q4以Q3的完整频率和振幅估计构造V1参数化响应模型，验证最优布局为\texttt{p105+p074+p062}。Q1/Q2处理单源故障工作表，Q3处理多源故障工作表；前者的2 Hz分量没有出现在后者中，因此四问体现方法的逐步扩展，不表示同一设备工况的复杂度逐级增加。

\subsection{噪声假设验证}
Q1/Q2和Q3残差的偏度、超额峰度和自相关均接近零，直方图与高斯分布吻合。Q2残差的Jarque-Bera正态性检验$p=0.669$，Ljung-Box白噪声检验$p=0.418$（滞后100阶）；Q3残差的Jarque-Bera检验统计量$1.382$（$p=0.5010$），Ljung-Box检验$Q(100)=86.43$（$p=0.8312$），均未拒绝原假设。在$N=40001$的大样本下，微小的自相关也可能达到统计显著性，因此以偏度/峰度/自相关的量级而非p值作为主要判断依据。

\section{模型评价与改进}
\subsection{模型的优点}
\begin{itemize}
    \item [(1)] \textbf{虚警率可经验校准。} 蒙特卡洛门限将多频点联合扫描的虚警率校准至目标水平，这是传统工业方法所不具备的关键能力\upcite{antoni2006spectral,smith2019optimal}。
    \item [(2)] \textbf{多重验证体系。} SSA和窄带滤波从不同原理提供了一致性对照；残差GLRT扫描直接验证了模型的充分性；合成数据实验覆盖了广泛SNR条件。
    \item [(3)] \textbf{参数不确定性量化。} 基于局部协方差矩阵的多元正态抽样（$10000$次）给出了频率、振幅、相位的区间估计。
    \item [(4)] \textbf{自动分量数量确定。} BIC准则在Q3中自动确定分量数量；完整流程1000次纯噪声试验未产生最终伪分量，Wilson 95\%上界为$0.383\%$。
    \item [(5)] \textbf{辨识界限定量化。} 通过系统近频扫描确定了$T=400$ s、噪声标准差约$0.1$条件下的经验辨识界限。
\end{itemize}

\subsection{模型的不足}
\begin{itemize}
    \item [(1)] 假设噪声为独立同分布高斯白噪声。工业场景可能具有色噪声或非高斯脉冲特性。
    \item [(2)] 信号模型假设振幅和频率恒定，未考虑转速波动引起的频率时变。
    \item [(3)] Q3的SIC顺序检测存在误差传播问题——早期分量的估计误差传递给后续分量，且缺乏纠正机制。
    \item [(4)] 当前真实数据解在所测试的$\alpha$、$\Delta$BIC和$N_{\text{MC}}$范围内变化很小；该结论未覆盖低SNR检测边界。精修搜索半径$h=0.012$ Hz时出现错误局部极小。
    \item [(5)] 单通道数据无法唯一确定周期分量与物理故障源的对应关系。
\end{itemize}

\subsection{模型的推广方向}
\begin{itemize}
    \item [(1)] \textbf{非高斯/色噪声扩展：} 将白噪声方差替换为噪声协方差矩阵估计（AR模型拟合），或采用Darong等\upcite{darong2018incipient}的自适应滤波作为预白化前端。
    \item [(2)] \textbf{时变频率追踪：} 引入短时傅里叶变换，将GLRT统计量扩展至时间-频率平面，适应非稳态工况。
    \item [(3)] \textbf{SIC替代方案：} 考虑LASSO（频率字典上一次稀疏回归选择所有分量）或RELAX（循环精化）以减少误差传播。
    \item [(4)] \textbf{谐波族模型：} 引入$f_{k,h}=h\cdot f_k$约束，用BIC比较独立频率模型与谐波族模型，利用频率间的倍频关系提高分量归因的物理可解释性。
\end{itemize}
